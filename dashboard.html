DASHBOARDS

Emprendimiento Dos Puntos

Dos Puntos es un emprendimiento cuya principal labor es la venta de detalles a crochet hechos a mano a traves de instagram. Con la finalidad de digitalizar aún más el empredimiento, se diseñó una landing page donde se muestran los productos que ofrecen y que tiene como finalidad el llenado de un formulario de cotización. El dashboard se creó con la finalidad de medir las principales métricas de la landing page.

Para el desarrollo de este proyecto se realizó lo siguiente:
1. Conexión con Tag Manager y Google Analytics 4
2. Creación de los activadore y etiquetas
3. Conexión con Big Query y creación de la tabla
4. Exportación de los datos a Locker Studio y creación del dashboard

Bootcamp Hackspace

Hackspace es un bootcamp cuya principal labor es la de preparar a profesionales en las disciplinas tecnológicas de mayor demanda en el pasi como Data Analytics, Data Science, etc. El dashboard se creó con la finalidad de medir las principales métricas de la landing page y conocer más el entorno de Locker Studio.

Para el desarrollo de este proyecto se realizó lo siguiente:
1. Se estudió el diccionario de la información recopilada de la página
2. Exportación de los datos a Locker Studio
3. Acondicionamiento de los datos y cambio de tipo de las columnas.
4. Exploración de los datos y creación del las gráficas
