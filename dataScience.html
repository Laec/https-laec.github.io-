Predicción de la productividad

En este proyecto se utilizó algoritmos de inteligencia artificial para analizar datos históricos y prever la productividad futura en una empresa textil. Esto se logra alimentando el modelo de aprendizaje automático con datos relevantes sobre factores que afectan la productividad, como el personal, el equipo, los recursos y las metas de productividad. El modelo luego utiliza esta información para hacer predicciones precisas sobre la productividad futura.

Para el desarrollo de este proyecto se realizó lo siguiente:
1. Limpieza de los datos atipicos y faltantes
2. Exploración de los datos usando Plotly
3. Transformación(Label encoding, OHE) y normalización de datos
4. Evaluación de rendimiento de modelos(Linear Regression, ANN)

Topic modelling en Publicaciones científicas

El topic modeling con BERT en publicaciones científicas es un enfoque para clasificar y agrupar automáticamente documentos científicos en temas relevantes utilizando la tecnología BERT (Bidirectional Encoder Representations from Transformers). Esto permite a los investigadores tener una comprensión más profunda de los temas importantes dentro de una determinada área de estudio, y puede ser útil para identificar tendencias y patrones en la investigación.
