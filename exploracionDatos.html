EXPLORACIÓN DE DATOS

Ventas de KFC

En este proyecto se realizó el proceso de examinar y analizar grandes cantidades de información sobre las ventas de KFC con el objetivo de descubrir patrones y tendencias útiles. Esto puede incluir análisis de factores como el rendimiento de las ventas en diferentes ubicaciones, las preferencias de los clientes en cuanto a los productos y servicios ofrecidos por KFC. El objetivo de la exploración de datos en las ventas de KFC es ayudar a la empresa a tomar decisiones informadas sobre cómo mejorar y optimizar sus operaciones y aumentar sus ventas.

Para el desarrollo de este proyecto se realizó lo siguiente:
1. Limpieza de los datos atipicos y faltantes con árbol de decisiones
2. Exploración de los datos usando pandas
3. Creación de visualizaciones personzalizadas con plotly

Precio de casas

Se desarrollo el análisis y visualización de los datos relacionados con los precios de venta de casas en King County con el fin de entender la relación entre diferentes variables y los precios de venta de las casas. Este análisis incluye la identificación de tendencias y patrones, la construcción de modelos predictivos para estimar los precios futuros y la identificación de factores que influyen en los precios de las casas. El objetivo es utilizar los resultados de la exploración de datos para mejorar la comprensión del mercado inmobiliario en King County.

Para el desarrollo de este proyecto se realizó lo siguiente:
1. Limpieza de los datos atipicos y faltantes
2. Exploración de los datos usando Plotly
3. Transformación(Label encoding, OHE, skew correction) de datos
4. Normalización MinMax de datos
5. Evaluación de rendimiento de modelos(Linear Regression, ANN)
